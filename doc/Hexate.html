    hexate v0.5.1 Hexate

    A simple module to convert to and from hex encoded strings.

    Encodes / decodes both lists and binaries.

    decode(hex_str)

    Returns a decoded binary from a hex string in either list or binary form.

    Examples

    iex> Hexate.decode("54686973206973206120746573742e")
"This is a test."

iex> Hexate.decode('54686973206973206120746573742e')
"This is a test."

    encode(int, digits \\ 1)

    Returns a hex encoded binary from a list, binary or integer.

    Examples

    iex> Hexate.encode("This is a test.")
"54686973206973206120746573742e"

iex> Hexate.encode('This is a test.')
"54686973206973206120746573742e"

iex> Hexate.encode(123456)
"1e240"

iex> Hexate.encode(15, 4)
"000f"

iex> Hexate.encode(15.0, 2)
"0f"

iex> Hexate.encode(15.0)
"f"
